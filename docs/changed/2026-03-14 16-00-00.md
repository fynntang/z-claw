# 变更记录：UI 重新设计（现代感 + AI Agent 风格）

**时间**: 2026-03-14  
**类型**: UI/UX 重设计

---

## 设计方向

- **调性**: 现代、克制、偏产品级 AI 助手 — 深色底、弱渐变氛围、单一强调色（青色/cyan）。
- **字体**: Plus Jakarta Sans Variable 作为主 UI 字体，JetBrains Mono 保留用于代码/技术场景。
- **色彩**: 背景 `#0a0a0c`，表面层级 `#111113` / `#18181b`，强调色 `#22d3ee`，边框与文字使用设计变量统一。

---

## 修改摘要

### 1. 设计系统 `src/routes/layout.css`
- 新增 ZClaw 设计变量：`--app-font-sans` / `--app-font-mono`、`--app-bg`、`--app-bg-mesh`（极弱径向渐变）、`--app-surface` / `--app-surface-elevated`、`--app-accent` 及 `--app-accent-muted` / `--app-accent-soft`、`--app-text` / `--app-text-secondary` / `--app-text-muted`、`--app-border` / `--app-border-focus`、`--app-radius*`。
- 引入 `@fontsource-variable/plus-jakarta-sans`，body 使用 `var(--app-font-sans)`。
- 新增工具类：`.zclaw-app`（主容器+背景渐变）、`.zclaw-surface` / `.zclaw-surface-elevated`（卡片表面）。
- 暗色模式下 `--bg` / `--text` 等与 `--app-*` 对齐；滚动条使用设计变量。

### 2. 布局 `src/routes/+layout.svelte`
- 根容器增加 `zclaw-app`，应用新背景与字体。

### 3. 主页面 `src/routes/+page.svelte`
- 主内容区使用 `zclaw-surface` 替代硬编码背景与圆角。

### 4. TitleBar `src/lib/components/TitleBar.svelte`
- 毛玻璃效果：半透明背景 + `backdrop-filter: blur(12px)`，底边 `--app-border`。
- Logo 使用 `--app-accent` 背景与深色文字；状态与按钮使用设计变量，hover 为 accent 色。

### 5. AgentSidebar `src/lib/components/AgentSidebar.svelte`
- 使用 `zclaw-surface`；Agent 头像圆角改为 `rounded-xl`，选中态为 `ring-2 ring-[var(--app-accent)]`。
- 添加 Agent、设置、帮助按钮使用 `--app-surface-hover` / `--app-accent` 交互态。

### 6. ChannelSidebar `src/lib/components/ChannelSidebar.svelte`
- 使用 `zclaw-surface`；标题区「对话」+ 新对话按钮。
- 会话/频道项选中态为 `--app-accent-soft` + `border-[var(--app-accent)]/30`，图标与文字使用设计变量。

### 7. ChatArea `src/lib/components/ChatArea.svelte`
- 主区背景 `--app-surface-elevated`，顶栏边框与按钮使用设计变量，选中 Tab 为 `--app-accent-soft` + accent 文字。
- 空状态：居中图标+简短引导「向 xxx 发送第一条消息」与「Enter 发送，Shift+Enter 换行」。
- 消息气泡：助手为 surface+边框，用户为 `--app-accent` 背景+深色字；发送中状态样式统一。
- 输入区：`chat-input-wrap` 使用 surface 与 focus 时 `--app-border-focus` 环；发送按钮为 accent 实心。

### 8. RightPanel `src/lib/components/RightPanel.svelte`
- 背景与边框改为 `--app-surface` / `--app-border`；区块标题改为小写+字间距；卡片使用 `--app-surface-elevated` 与设计变量文字层级。
- 文件列表项与「上传新文件」hover 使用 `--app-accent-soft` / accent 色。

### 9. 依赖
- 新增 `@fontsource-variable/plus-jakarta-sans@^5.2.8`（devDependencies）。

---

## 可执行与验证

- `pnpm install` 后 `pnpm tauri dev` 可正常跑；字体与 CSS 变量在暗色主题下全局生效。
- 建议在真机/多分辨率下检查 TitleBar 毛玻璃与侧栏折叠、Chat 空状态与气泡布局。

---

## 后续建议

- 亮色主题：在 `.dark` 外为 `:root` 或 light 类补充 `--app-*` 浅色取值，保持同一套变量名。
- 动效：可为侧栏展开/收起、Tab 切换增加短时长 transition，与现有 transition 类统一。
